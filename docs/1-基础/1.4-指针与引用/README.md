# 指针

**定义**：指针是一个变量，其值为另一个变量的地址，即，内存位置的直接地址。

**形式**：

	- 声明：type *var-name; 
	  例如：int i = 10;
	        int *pNum = &i;
	- 取值：*var-name;
	  例如: *pNum;

**大小**：4字节（32位程序），8字节（64位程序）

# 引用

**定义**：引用变量是一个别名，也就是说，它是某个已存在变量的另一个名字。一旦把引用初始化为某个变量，就可以使用该引用名称或变量名称来指向变量。

**形式**：

- 声明：type &var-name = var;

  例如：int &refNum = i;

**大小**：原类型大小，sizeof(type);
