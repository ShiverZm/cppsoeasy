# 写个宏定义一年有多少秒

## 标答

### 考虑闰年

闰年两种情况：

1. 非世纪年能被4整除，且不能被100整除的是闰年。（如2004年是闰年，1901年不是闰年）

2. 世纪年能被400整除的是闰年。（如2000年是闰年，1900年不是闰年）

#define Seconds_Of_Year(year) {  (((( (year % 4==0) && (year % 100 != 0 )) || (year % 400==0)) ? 1 : 0 )+365) * 24 * 3600 }

### 不考虑闰年

注意：UL表示长整型 考细节

#define S (60 * 60 * 24 * 365UL)
